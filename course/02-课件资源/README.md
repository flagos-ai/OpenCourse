# 课件资源

本目录包含 FlagOS 高校课程的完整课件资源，包括中文版和英文版。

## 目录结构

```
课件资源/
├── 中文版/          # 完整中文课件（讲义、作业、视频索引）
│   ├── AI大模型通识课（入门课程）/    # 入门课程，非正式模块
│   ├── 模块1-AI系统软件基础与异构计算/
│   ├── 模块2-高性能AI算子与算子工程/
│   ├── 模块3-AI编译器原理与优化/
│   ├── 模块4-分布式并行训练与通信/
│   ├── 模块5-性能评测与下一代内核生成/
│   └── 完整课程汇总-48课时/    # 完整的48课时课程资源
└── 英文版/          # 英文课件索引
```

## 中文版

中文版课件包含入门课程、5 大模块和完整的 48 课时课程。

### 入门课程

| 课程 | 目录 | 说明 |
|------|------|------|
| AI 大模型通识课 | [AI大模型通识课（入门课程）](中文版/AI大模型通识课（入门课程）/) | 非正式模块，建立大模型整体认知 |

### 模块化课程

| 模块 | 目录 | 说明 |
|------|------|------|
| 模块 1 | [模块1-AI系统软件基础与异构计算](中文版/模块1-AI系统软件基础与异构计算/) | 硬件架构、SIMT 编程、Triton 基础 |
| 模块 2 | [模块2-高性能AI算子与算子工程](中文版/模块2-高性能AI算子与算子工程/) | Attention、算子融合、Profiling |
| 模块 3 | [模块3-AI编译器原理与优化](中文版/模块3-AI编译器原理与优化/) | IR、MLIR、Pass 优化 |
| 模块 4 | [模块4-分布式并行训练与通信](中文版/模块4-分布式并行训练与通信/) | 数据/模型并行、通信优化 |
| 模块 5 | [模块5-性能评测与下一代内核生成](中文版/模块5-性能评测与下一代内核生成/) | 性能分析、自动调优 |

### 完整课程

| 课程 | 目录 | 说明 |
|------|------|------|
| 48课时完整课程 | [完整课程汇总-48课时](中文版/完整课程汇总-48课时/) | 包含全部48课时的讲义、作业、视频索引 |

## 使用方式

### 教师使用

1. 根据教学需要选择对应模块
2. 下载讲义进行教学
3. 布置作业给学生
4. 参考视频资源进行备课

### 学生使用

1. 下载讲义预习和复习
2. 完成作业巩固知识
3. 观看视频补充学习
4. **参与 [FlagOS 算子大赛](https://flagos.io/events?tab=competition)**：学以致用，赢取奖金，竞赛成绩可作为学习成就
