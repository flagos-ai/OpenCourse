# 实验材料包

本目录汇总 FlagOS 课程的实验资源，帮助高校开展实践教学。

## 重要说明

⚠️ **受限于现有资源，目前的实验不一定与课件资源完全对应。** 

各高校可以：
- 使用我们提供的现有实验
- 根据课程需要设计自己的实验

无论选择哪种方式，我们都将提供相应的**算力支持**。

## 文件说明

| 文件 | 说明 |
|------|------|
| [现有实验列表.md](现有实验列表.md) | 所有现有实验的 Gitee 仓库链接 |
| [实验环境说明.md](实验环境说明.md) | 基于 FlagOS 在线实验室的环境说明 |
| [自定义实验支持.md](自定义实验支持.md) | 高校自定义实验的算力支持申请流程 |

## 实验概览

| 实验 | 内容 | 支持平台 |
|------|------|----------|
| Day 1 | GPU/NPU 编程基础、Triton 内核编程 | 天数智芯、华为 Ascend |
| Day 2 | 性能调优、Buffer 模型分析、自动调优 | 天数智芯、华为 Ascend |
| Day 3 | vLLM 大模型部署、FlagGems 集成 | 天数智芯、华为 Ascend |

## 使用方式

### 使用现有实验

1. 查看 [现有实验列表](现有实验列表.md) 选择适合的实验
2. 参考 [实验环境说明](实验环境说明.md) 配置实验环境
3. 克隆 Gitee 仓库开始实验

### 自定义实验

1. 设计实验内容和要求
2. 参考 [自定义实验支持](自定义实验支持.md) 申请算力支持
3. 我们将协助配置实验环境

## 实验平台

推荐使用 **FlagOS 在线实验室** 进行实验，无需本地配置环境。

详见 [在线实验室指南](../07-在线实验室指南/)。
