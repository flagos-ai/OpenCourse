# 高校教育资源

本目录为 FlagOS 高校合作支持中心，提供完整的课程资源、实验材料和支持服务，帮助中国高校便捷接入 FlagOS 课程体系。

## 核心优势

- **零硬件投入**：FlagOS 提供在线实验室和 GPU/NPU 算力支持，高校无需自建计算环境
- **完整课程体系**：48 课时模块化课程，从 AI 硬件基础到分布式训练全栈覆盖
- **灵活接入方式**：教师可选取部分内容融入课程，或系统性引入完整课程

## 快速开始

### 高校教师

1. 了解 [项目背景与意义](01-项目背景与意义/)
2. 查看 [课件资源](02-课件资源/) 选择适合的内容
3. 参考 [教师培训指南](05-教师培训指南/) 了解如何融入课程

### 教务管理人员

1. 查看 [课程大纲示例](03-课程大纲示例/) 了解课程设计
2. 了解 [在线实验室](07-在线实验室指南/) 的算力支持
3. 通过 [高校合作流程](08-高校合作流程/) 申请合作

### 学生

1. 通过 [在线实验室指南](07-在线实验室指南/) 获取计算资源
2. 查看 [实验材料包](04-实验材料包/) 进行实践
3. 参与 [算子大赛](06-学生学习成就/FlagOS算子大赛.md) 学以致用
