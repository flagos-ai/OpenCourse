## 合作模式

我们提供灵活的合作模式，满足不同高校的需求。

### 模式一：课程融入

教师选取部分模块内容融入现有课程。

- 选择适合的教学内容
- 自行设计考核方式
- 我们提供算力支持

### 模式二：完整引入

系统性引入 FlagOS 完整课程体系。

- 采用完整课程设计
- 使用全部教学资源
- 可申请教师培训

### 模式三：合作共建

深度合作共建课程。

- 共同开发教学内容
- 联合培养学生
- 开展科研合作

## 申请流程

```
需求沟通 → 方案确定 → 签订协议 → 资源配置 → 正式开课
```

### 步骤 1：需求沟通

联系我们，说明课程引入意向和需求。

### 步骤 2：方案确定

根据沟通结果，确定合作方案：

- 课程内容选择
- 教学资源提供
- 算力配额分配
- 培训安排（如需）

### 步骤 3：签订协议

双方签订合作协议，主要约定：

- **非营利性**：课程资源仅供教学使用，不得用于商业目的
- **资源使用**：计算资源仅限选课学生使用，禁止挪作他用
- **知识产权**：课程内容的知识产权归属及使用范围
- **保密条款**：双方对合作中获知的敏感信息保密

### 步骤 4：资源配置

填写 [资源申请表](资源申请表.md)，我们将在 10~15 个工作日内：

- 开通在线实验室账号
- 提供教学资源
- 配置实验环境

### 步骤 5：正式开课

按计划开展教学，提供持续支持。

## 我们提供的支持

### 教学资源

- 完整的讲义和作业
- 实验材料和指导

### 算力支持

- 在线实验室账号

- 实验环境配置

### 培训支持
