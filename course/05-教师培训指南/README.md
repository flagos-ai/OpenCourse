## 两种融入方式

### 方式一：部分融入

教师可选取课件资源的部分内容融入自己的课程：

- 选择一个或多个模块
- 结合本校课程特点调整
- 自行设计实验内容
- 我们提供算力支持

### 方式二：完整引入

系统性引入 FlagOS 完整课程：

- 使用完整 48 学时课程体系
- 采用推荐的讲义、作业、实验
- 按推荐方式组织考核
- 可申请教师培训支持

## 教师培训方式

### 在线自学

- 通过课件和视频资源自学
- 参考课程大纲示例设计课程
- 有问题可通过社区或 Issues 咨询

### 线下培训

视情况组织集中线下培训：

- 课程内容讲解
- 实验操作指导
- 教学方法交流

详见 [线下培训说明.md](线下培训说明.md)。

## 学生成就机会

学习本课程的学生可以参加 **[FlagOS 算子大赛](https://flagos.io/events?tab=competition)**：

- 学以致用，将课程所学应用于真实项目
- 优秀选手可获得丰厚奖金
- 竞赛成绩可作为学习成就和项目经历
- 为求职简历增添亮点

详见 [学生学习成就](../06-学生学习成就/)。
