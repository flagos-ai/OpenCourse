# 平台使用手册

本文档提供 FlagOS 在线实验室的详细操作指南。

## 平台入口

🔗 **在线实验室**：https://flagos.io/OnlineLab

## 登录与界面

### 登录步骤

1. 访问 https://flagos.io/OnlineLab
2. 输入用户名和密码
3. 点击「登录」

### 主界面说明

登录后可见：

- **资源概览**：当前配额使用情况
- **实例管理**：创建和管理计算实例
- **文件管理**：上传和管理文件
- **帮助文档**：使用指南和 FAQ

## 创建实例

### 步骤 1：选择计算平台

根据实验需求选择：

| 平台 | 适用场景 |
|------|----------|
| 天数智芯 BI-V150 | Triton 编程、vLLM 部署 |
| 华为 Ascend 910C | NPU 编程、大模型训练 |

### 步骤 2：选择软件环境

选择预配置的环境：

- Triton 开发环境
- vLLM 部署环境
- 分布式训练环境

### 步骤 3：配置实例

设置实例参数：

- 实例名称
- 资源规格
- 存储挂载

### 步骤 4：启动实例

确认配置后点击「启动」，等待实例就绪。

## 使用实例

### Jupyter Notebook

实例启动后，可以：

- 创建和运行 Notebook
- 编写和调试代码
- 查看运行结果

### 终端访问

切换到终端模式：

- 执行命令行操作
- 运行 Python 脚本
- 安装额外软件包

### 文件操作

- 上传本地文件
- 下载实验结果
- 管理项目目录

## 实验操作

### 编写代码

支持多种方式：

- Jupyter Notebook 交互式编程
- 终端命令行执行
- 上传本地代码运行

### 运行实验

```bash
# 运行 Python 脚本
python lab_script.py

# 运行 Jupyter Notebook
jupyter notebook
```

### 查看结果

- 查看控制台输出
- 可视化结果展示
- 日志文件查看

## 数据管理

### 上传文件

1. 点击「文件管理」
2. 选择「上传文件」
3. 选择本地文件
4. 等待上传完成

### 下载文件

1. 选择要下载的文件
2. 点击「下载」
3. 保存到本地

### 同步代码

建议使用 Git 管理代码：

```bash
# 克隆仓库
git clone https://gitee.com/xxx/lab.git

# 提交更改
git add .
git commit -m "update"
git push
```

## 实例管理

### 查看状态

- 运行中
- 已停止
- 错误

### 停止实例

实验完成后请及时停止实例：

1. 选择运行中的实例
2. 点击「停止」
3. 确认停止

### 释放资源

不再需要的实例可以删除：

1. 停止实例
2. 点击「删除」
3. 确认删除

## 常见问题排查

### 无法启动实例

- 检查配额是否充足
- 检查是否有冲突配置
- 联系管理员协助

### 实例响应慢

- 检查资源使用情况
- 减少并发任务
- 选择更高规格实例

### 连接中断

- 检查网络连接
- 重新登录尝试
- 联系管理员

## 安全建议

- 定期修改密码
- 不要共享账号
- 不要上传敏感数据
- 遵守平台使用规范
