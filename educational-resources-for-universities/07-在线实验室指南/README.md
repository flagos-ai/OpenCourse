# 在线实验室指南

本目录提供 FlagOS 在线实验室的使用指南。

## 核心优势

FlagOS 提供**在线实验室和算力支持**，这是高校接入课程的核心优势：

- **零硬件投入**：高校无需采购 GPU/NPU 设备
- **即开即用**：学生登录即可开始实验
- **多平台支持**：支持天数智芯、华为 Ascend 等多种芯片
