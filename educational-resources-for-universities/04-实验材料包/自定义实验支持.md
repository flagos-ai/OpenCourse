# 自定义实验支持

FlagOS 支持高校根据自身教学需要设计实验，并提供相应的算力支持。

## 支持政策

我们为开展 FlagOS 课程的高校提供：

- **在线实验室资源**：GPU/NPU 计算资源配额
- **实验环境配置**：协助配置特定实验环境
- **技术支持**：实验过程中的技术咨询

## 申请流程

请填写 [资源申请表](../08-高校合作流程/资源申请表.md)，说明：

- 课程/实验名称
- 预计学生人数
- 实验内容和资源需求
- 课程时间安排

我们将在 5 个工作日内评估并回复。

## 实验设计建议

建议围绕以下方向设计实验：

1. **算子开发与优化**：Triton 内核编写、性能分析与调优
2. **编译器实践**：MLIR Pass 编写、算符融合实现
3. **分布式训练**：数据并行实现、模型并行部署
4. **大模型部署**：vLLM 服务搭建、模型量化与优化

## 注意事项

- 计算资源仅供课程实验使用
- 重要数据请及时保存到本地
- 实验完成后及时释放资源
