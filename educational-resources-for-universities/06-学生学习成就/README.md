# 学生学习成就

本目录介绍学生通过 FlagOS 课程学习可以获得的成就机会。

## FlagOS 算子大赛

学生在学习过程中可以报名参加 **FlagOS 算子大赛**，实现学以致用。

### 大赛链接

🔗 **大赛主页**：https://flagos.io/events?tab=competition

### 参赛优势

1. **学以致用**：将课程所学应用于真实项目
2. **赢取奖金**：优秀选手可获得丰厚奖金
3. **项目成就**：竞赛成绩可作为学习成就和项目经历
4. **就业加分**：为求职简历增添亮点

### 参赛方式

1. 访问大赛主页了解详情
2. 按要求报名参赛
3. 完成算子开发与优化
4. 提交作品参与评审

## 其他成就机会

### 开源贡献

学生可以向 FlagOS 开源项目贡献代码：

- FlagGems：https://github.com/flagos-ai/FlagGems
- FlagTree：https://github.com/flagos-ai/FlagTree
- FlagScale：https://github.com/flagos-ai/FlagScale

开源贡献经历是宝贵的项目经验，有助于提升就业竞争力。

### 实习机会

表现优秀的学生有机会获得：

- BAAI 实习机会
- AI 芯片公司实习推荐
- 开源社区运营经验

## 学习建议

### 技能提升路径

```
课程学习 → 实验练习 → 算子大赛 → 开源贡献 → 实习就业
```

### 建议时间安排

| 阶段 | 内容 | 时间 |
|------|------|------|
| 课程学习 | 完成模块学习 | 1 学期 |
| 实验练习 | 完成配套实验 | 课程期间 |
| 算子大赛 | 参加比赛 | 每年定期举办 |
| 开源贡献 | 持续贡献 | 长期 |
